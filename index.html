Raspberry pi 3b

De raspberry pi is heel eenvoudig gezegd een kleine computer. Om op deze "computer" te kunnen werken moeten we een toetsen bord en monitor aansluiten. Via een programma op de raspberry pi genaamd Weewx plaatsen we de ontvangen data door middel van plots, rapporten en HTML-pagina's op een site. Deze plots, rapporten en HTML-pagina's worden aangemaakt door Weewx. De Weewx is een programma dat geschreven is in Python. Het bestaat uit meer dan 15 000 regels code plus nog eens 15 000 regels voor alle ondersteunende Hardware. Als je programma werkt en het is in gebruik dan werkt Weewx in Daemon.

Test opstelling

Bij de test opstelling maken we gebruik van de arduino die in een seriële verbinding is met de raspberry pi. Hierdoor kunnen we de verbinding van de buitenmodule module testen van arduino naar arduino. Maar ook al voor de eerste keer een Python programma testen. Deze opstelling was succesvol. De verzonden data van de buitenmodule kwam correct op het scherm aangesloten op de raspberry pi.
